Documentation: Laplace Edge Detection

Description

This action performs edge detection using a 3x3 Laplace mask. Greyscale intensity is calculated as the average value of the RGB triplet.

Example usage

var img = new Image();
img.onload = function() {
	Pixastic.process(img, "laplace", {edgeStrength:0.5,invert:false,greyLevel:0});
}
document.body.appendChild(img);
img.src = "myimage.jpg";

Parameters

  • greyLevel (int)
    Level of grey added to image.
  • edgeStrength (float)
    A positive value indicating the “density” of the edges.
  • invert (bool)
    Set to true to invert the final image.
