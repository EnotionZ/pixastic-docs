Documentation: Blur Fast

Description

A faster blur with variable strength. Rather than do a per-pixel manipulation like the regular Blur action, this one achieves a blur-like effect by rescaling the image a small amount over and over again.

Example usage

var img = new Image();
img.onload = function() {
	Pixastic.process(img, "blurfast", {amount:0.5});
}
document.body.appendChild(img);
img.src = "myimage.jpg";

Parameters

  • amount (float)
    The strength of the blur effect. Value between 0 and 5.

Known issues

  • As expected, this action takes longer to perform for strong blur effects while gentle blurring is rather speedy. Not really an "issue" as such, but worth noting.
