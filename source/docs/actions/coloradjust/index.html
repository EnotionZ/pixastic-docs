Documentation: Color Adjust

Description

This action shifts the red, green and blue channels of the image by an amount specified by you.

Example usage

var img = new Image();
img.onload = function() {
	Pixastic.process(img, "coloradjust", {red:0.5,green:0,blue:0});
}
document.body.appendChild(img);
img.src = "myimage.jpg";

Parameters

  • red (float)
    Amount to shift red, value from -1 to 1.
  • green (float)
    Amount to shift green, value from -1 to 1.
  • blue (float)
    Amount to shift blue, value from -1 to 1.
