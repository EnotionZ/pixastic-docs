Documentation: Emboss

Description

This creates a simple emboss effect. It is possible vary the brightness of the image and select the direction of the light from a predefined list (top left, top, top right, etc..).

Example usage

var img = new Image();
img.onload = function() {
	Pixastic.process(img, "emboss", {greyLevel:100,direction:"topright"});
}
document.body.appendChild(img);
img.src = "myimage.jpg";

Parameters

  • strength (float)
    The strength of the effect. Value between 0 and 10.
  • greyLevel (int)
    The standard level of grey in the final image. A value between 0 and 255. Defaults to 180.
  • direction (string)
    The "direction" of the light source. One of "topleft", "top", "topright", "right", "bottomright", "bottom", "bottomleft" or "left".
  • blend (bool)
    If true, the embossed image will be blended with the original image rather than producing a greyscale image.
