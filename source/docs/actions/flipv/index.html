Documentation: Flip Vertically

Description

This action simply flips the image vertically.

  • Demo
  • Example code
  • Parameters

Demo


Example usage

var img = new Image();
img.onload = function() {
	Pixastic.process(img, "flipv");
}
document.body.appendChild(img);
img.src = "myimage.jpg";

Parameters

This action has no parameters.
