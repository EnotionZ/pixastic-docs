Documentation: Noise

Description

Adds random noise to the image. Choose from either coloured or monochrome noise.

Example usage

var img = new Image();
img.onload = function() {
	Pixastic.process(img, "noise", {mono:true,amount:0.5,strength:0.5});
}
document.body.appendChild(img);
img.src = "myimage.jpg";

Parameters

  • amount (float)
    The amount of noise added to the image. Value between 0 and 1, ie. a value of 0.5 means that every other pixel is affected.
  • strength (float)
    The strength of the noise. Value between 0 and 1.
  • mono (bool)
    Set to true to enable greyscale/monochromatic noise.
