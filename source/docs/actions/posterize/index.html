Documentation: Posterize

Description

This action reduces the number of colours to a specified number of levels, creating a posterize effect.

  • Demo
  • Example code
  • Parameters

Demo

Levels:

Example usage

var img = new Image();
img.onload = function() {
	Pixastic.process(img, "posterize", {levels:10});
}
document.body.appendChild(img);
img.src = "myimage.jpg";

Parameters

  • levels (int)
    Number of levels, 0 to 255
