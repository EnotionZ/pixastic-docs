Documentation: Sharpen

Description

This action applies a 3x3 sharpen kernel on the image, sharpening it a bit. The strength of the sharpening effect can be controlled with the amount parameter. See also the Unsharp Mask action.

Example usage

var img = new Image();
img.onload = function() {
	Pixastic.process(img, "sharpen", {
		amount : $("#value-amount").val()
	});
}
document.body.appendChild(img);
img.src = "myimage.jpg";

Parameters

  • amount (float)
    The amount of sharpening. Value between 0 and 1.
